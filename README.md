# to_gcode

MVP para convertir archivos SVG a formato GCode para plotters, diseñado bajo principios de Arquitectura Limpia y DDD.

## Funcionalidades
- Configuración de parámetros de máquina (dimensiones, comandos pen-up/down).
- Conversión de archivos SVG locales.
- Conversión de SVG desde URL (procesamiento en backend).
- Modo Test: Genera solo las primeras 100 líneas de GCode.

## Ejecución
Requiere entorno virtual y dependencias instaladas.
./run.sh
Abre frontend/index.html en tu navegador.
